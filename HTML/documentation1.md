## Introduction 
### HTML 
1. Describes the dtructure of the web 
2. Consists of a series of elements

### Why?
1. To understand how the frontend works
2. To build and test APIs easily
3. To debug issues between frontend & backend

### Text and Content 
- Heading 
`<h1></h1?>`
- Paragraph
`<p></p>`
- Image 
`<img src="..." alt="...">`
- Video
`<audio controls>`

### Lists 
- Unorderd 
`<ul>` with `<li>`
- Ordered 
`<ol>` with `<li>`
- Description 
`dl>` `<dd>` `<dt>`

### Tables 
- Each table row starts with a `<tr>` and ends with a `</tr>` tag.
- Each table cell is defined by a `<td>` and a `</td>` tag.
- For table header cells we use `<th>` instead of `<td>`.

```
<table>
  <tr>
    <th>Name</th>
    <th>Age</th>
  </tr>
  <tr>
    <td>Jana</td>
    <td>20</td>
  </tr>
</table>
```
